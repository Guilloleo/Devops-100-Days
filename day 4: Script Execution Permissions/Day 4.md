# Day 4: Script Execution Permission

```
Para ejecutar el archivo dejado en el servidor app3 hacemos lo siguiente

1. Conectamos con el servidor que nos indica el ejercicio
2. Con el comando hostname verificamos en que servidor estamos contecdo
3. Con el comando ls -l /tmp/xfusioncorp.sh verificamos que el archivo se encuentre en la ruta.
4. Con el coamdno sudo chmod 755 /tmp/xfusioncorp.sh, otorgamos permisos de ejecucion.
5. vamos a la ruta donde esta el archivo y nos debe permitir ver lo que contiene el archivo.
```
